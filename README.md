# Cooper

LMS (Learning Management System) built with Next.js, Supabase, and Tailwind CSS.

## Features

- **Student view** — enroll in courses, follow lessons, mark lessons complete
- **Instructor view** — create and publish courses, manage modules/lessons, enroll students
- **Auth** — SSO Google with Supabase
- **Dark mode** — via `next-themes`

## Stack

| Layer | Tech |
|---|---|
| Framework | Next.js (App Router) |
| Database / Auth | Supabase |
| Styling | Tailwind CSS + shadcn/ui (Radix) |
| Language | TypeScript |
